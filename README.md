# 🎬 Movie Recommender System

A Machine Learning based Movie Recommendation System that suggests 
top 5 similar movies based on your selection.

## 🚀 Live Demo
👉 [Click here to try the app](https://movie-recommender-6znknascsyz9rfzt7dzzfa.streamlit.app/)

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy
- TMDB 5000 Movie Dataset

## 📌 Features
- Content-based filtering using Cosine Similarity
- Processes 5,000+ movies from TMDB dataset
- Real-time top-5 movie recommendations
- Clean and interactive web interface

## 🔍 How It Works
1. Select any movie from the dropdown
2. Click **Recommend**
3. Get top 5 similar movies instantly

## 📊 Algorithm
- Text vectorization using CountVectorizer
- Cosine Similarity to find closest matching movies
- Features used: genres, keywords, cast, crew, overview
